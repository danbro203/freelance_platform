# Биржа фриланса

Веб-платформа для заказчиков и исполнителей на Flask.

## Возможности
- Регистрация заказчиков и работников
- Создание заказов с прикреплёнными файлами
- Отклики на заказы с ценой
- Система рейтингов и отзывов
- Поиск и фильтрация заказов

## Установка
```bash
pip install -r requirements.txt
python app.py
```

## Технологии
- Python 3.12
- Flask 3.0
- SQLAlchemy
- Bootstrap 5
